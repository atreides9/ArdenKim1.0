Hanacard Main Feature

Redesigning the core features of Hanacard mobile app to improve financial management experience with better visualization and simplified transaction flows.

Project Overview

This project focused on redesigning the main features of Hanacard's mobile application to provide users with better financial insights, simplified card management, and improved transaction monitoring capabilities.

Role

UX/UI Designer

Duration

5 weeks

Team

Solo Project

Tools

Figma, Adobe XD

Problem Statement

Users found it difficult to track their spending patterns and manage multiple cards effectively. The existing interface lacked clear visual hierarchy and intuitive navigation for financial data.

Solution

Created a comprehensive redesign focusing on data visualization, simplified card switching, and improved financial insights. The new design emphasizes clarity and helps users make informed financial decisions.

Key Features

  • Enhanced spending analytics with visual charts
  • Simplified multi-card management interface
  • Real-time transaction notifications
  • Personalized financial insights and tips
  • Streamlined bill payment process
